# OceanMate-iOS

SwiftUI iOS app for species-based fishing recommendations with a clean dark-mode UI.

## Features
- Search and select fish species
- Recommended bait + rigs + techniques per species
- “Copy Loadout” to clipboard
- Built-in chat-style fishing assistant (mock/offline)
